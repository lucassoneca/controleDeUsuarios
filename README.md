# Controle de Usuários 

Este projeto é um sistema simples de gerenciamento de usuários, desenvolvido exclusivamente para fins de estudo e aprendizado. O objetivo principal foi colocar em prática conceitos de desenvolvimento, estruturação de projetos e manipulação de dados.


Sobre o Projeto

O Controle de Usuários permite realizar operações básicas de gestão (como criação, leitura, atualização e exclusão - CRUD). Através deste projeto, pude aprofundar meus conhecimentos em:

- Estruturação de pastas e organização de código.
- Lógica de programação aplicada a sistemas de cadastro.
- Construtores
- Loops de repetição
- Controladores de acesso
- Estruturas condicionais
- Sobreescrita de métodos
- Estrutura de dados

Nota: Este repositório foi criado para questões de estudo. Portanto, não deve ser considerado um modelo para ambientes de produção, mas sim um registro da minha evolução técnica.
